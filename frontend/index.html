JSON WORKBENCH

从粘贴数据到定位问题，一页完成

JSONUtils 把格式化、语法校验、智能修复和结构分析放进同一个工作区。你可以直接粘贴接口响应、日志片段或配置文件；普通格式化、校验与查询在当前浏览器内完成，无需先注册或上传文件。

格式化与压缩 JSON

把单行 JSON 美化成可读层级，或压缩成适合请求参数和测试数据的紧凑文本；嵌套字符串还可继续深度展开。

校验并定位语法错误

检查缺少逗号、引号和括号不匹配等问题，再回到具体位置修正；语法通过后可继续做 Schema 约束检查。

预览并修复异常 JSON

修正常见的单引号、尾随逗号或键名缺少引号，同时保留人工复核环节，避免把“能解析”误当成“业务值正确”。

查询、对比与生成类型

使用 JSONPath 提取嵌套字段，对比两份 JSON 的结构变化，并从样例生成 TypeScript 类型或执行 JSON Schema 校验。

需要具体输入和预期结果？查看 JSONUtils 使用指南、JSON Diff、JSON Schema 与 JSON 转 TypeScript。
